# Newsletter Template Generator

A web-based application for educational institutions to design, manage, and publish professional newsletters easily. It provides a robust, intuitive, and highly customizable interface that allows editorial teams to effortlessly design, manage, and publish professional-grade newsletters.

## Project info

This project is built with:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

Follow these steps to run the application locally:

```sh
# Step 1: Clone the repository
git clone git@github.com:Kavinjr1612/Newsletter-Template-Generator.git

# Step 2: Navigate to the project directory
cd Newsletter-Template-Generator

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Features
- **Dynamic Theming:** Support for customizable color palettes, background, and typography.
- **Pre-configured Templates:** Choose from different styles suited for modern corporate or newspaper layouts.
- **Export to PDF:** Generates high-fidelity PDF outputs.
- **Real-time Preview:** Split-screen layout displaying data management on one side and real-time template preview on the other.
